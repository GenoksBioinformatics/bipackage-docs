__Usage__

```bash
bip fastqvalidate [-h] --dir DIR
```

__Options__

| Option      | Alias | Required | Description                                      |
|-------------|-------|----------|--------------------------------------------------|
| `-h, --help`|       | No       | Show this help message and exit                 |
| `--dir DIR` | `-d`  | Yes      | Directory to perform the validation of fastq files.
