# Welcome to BIPackage Docs

## Installation

### Install via pip

```shell
pip install bipackage
```
### Install via pipx (suggested)

```shell
pipx install bipackage
```

## Usage

### In Python

```python
import bipackage as bip
```

### CLI

BIP

```shell
bip --help
```

BIPIPE

```shell
bipipe --help
```
