## USAGE


```bash
bipipe get-config [-h] [configs ...]
```

__Positional Arguments__

| Argument  | Required | Description |
|-----------|----------|-------------|
| `configs` | No       | Configuration files to retrieve |

__Options__

| Option      | Required | Description                      |
|-------------|----------|----------------------------------|
| `-h, --help`| No       | Show this help message and exit
